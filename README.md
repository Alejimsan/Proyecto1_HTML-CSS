# Proyecto 1: HTML y CSS - Fundamentos de Diseño Web

Este repositorio contiene la solución práctica al primer proyecto enfocado exclusivamente en el desarrollo de interfaces web utilizando HTML5 y CSS3. El objetivo principal es transformar un diseño o estructura básica en una página web visualmente atractiva, aplicando estilos, maquetación y reglas de diseño modernas.

## Descripción del Proyecto

El proyecto consiste en el desarrollo de una página web estática donde se pone en práctica la separación definitiva entre el contenido (HTML) y la presentación (CSS). Se ha trabajado especialmente en el posicionamiento de elementos, la tipografía, el uso de colores y la creación de una interfaz de usuario coherente.

## Estructura del Proyecto

El repositorio está organizado de la siguiente manera para facilitar su mantenimiento:

* index.html: Archivo principal con la estructura semántica de la web.
* assets/: Carpeta que contiene los recursos estáticos.
    * css/: Directorio con las hojas de estilo (style.css).
    * img/: Imágenes, iconos y logotipos utilizados.
* docs/: Documentación del diseño o bocetos previos.

## Tecnologías Utilizadas

* HTML5: Estructuración semántica de la información.
* CSS3: Aplicación de estilos, transiciones y maquetación.
* Metodología BEM: (Si se ha utilizado) Para una nomenclatura de clases clara y escalable.

## Conceptos Trabajados

En este proyecto se han implementado y dominado los siguientes conceptos técnicos:

### Maquetación y Posicionamiento
* Box Model: Gestión de márgenes, rellenos, bordes y dimensiones.
* Flexbox: Uso de contenedores flexibles para alinear y distribuir elementos de forma eficiente.
* Posicionamiento: Uso de propiedades como relative, absolute y fixed.

### Estilos Avanzados
* Selectores CSS: Uso de selectores de clase, ID, descendientes y pseudoclases (:hover, :active).
* Tipografía: Integración de fuentes externas (Google Fonts) y propiedades de texto.
* Fondos y Bordes: Uso de gradientes, imágenes de fondo y bordes redondeados.

### Diseño Responsivo
* Media Queries: Adaptación básica del diseño para diferentes tamaños de pantalla (móvil, tablet y escritorio).

## Cómo visualizar el proyecto

1. Clona el repositorio en tu máquina local.
2. Accede a la carpeta del proyecto.
3. Abre el archivo index.html en tu navegador.
